# RSS-to-Email
An RSS client that monitors your feeds and sends new posts straight to your email inbox.

## Features
- Makes your subscriptions easy to manage with Newsletter-Subscriptions-As-Code
- Simple Deployment with Docker

## Technologies
- Python
- `feedparser`
- `pyyaml`
- Docker
